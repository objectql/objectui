The Scroll Area component provides a scrollable container with customized scrollbars.

## Basic Usage

<ComponentDemo
  schema={{
    type: 'scroll-area',
    height: '200px',
    content: {
      type: 'flex',
      direction: 'column',
      gap: 2,
      children: Array.from({ length: 20 }, (_, i) => ({
        type: 'text',
        value: `Item ${i + 1}`
      }))
    }
  }}
  title="Vertical Scroll"
/>

## Horizontal Scroll

<ComponentDemo
  schema={{
    type: 'scroll-area',
    width: '400px',
    orientation: 'horizontal',
    content: {
      type: 'flex',
      gap: 2,
      children: Array.from({ length: 20 }, (_, i) => ({
        type: 'badge',
        text: `Tag ${i + 1}`
      }))
    }
  }}
  title="Horizontal Scroll"
/>

## Schema

```typescript
interface ScrollAreaSchema {
  type: 'scroll-area';
  content: ComponentSchema;       // Scrollable content
  height?: string;                // Container height
  width?: string;                 // Container width
  orientation?: 'vertical' | 'horizontal';
  className?: string;
}
```
