The Toggle component is a two-state button that can be toggled on or off.

## Basic Usage

  schema={{
    type: 'toggle',
    label: 'Bold',
    icon: 'bold'
  }}
  title="Basic Toggle"

## Schema

```typescript
interface ToggleSchema {
  type: 'toggle';
  label?: string;                 // Toggle label
  icon?: string;                  // Lucide icon name
  variant?: 'default' | 'outline';
  size?: 'sm' | 'default' | 'lg';
  pressed?: boolean;              // Toggle state
  
  // Events
  onPressedChange?: string | ActionConfig;
  
  // States
  disabled?: boolean;
  
  // Styling
  className?: string;
}
```
