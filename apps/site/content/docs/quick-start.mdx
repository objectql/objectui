# Quick Start

This guide will help you get started with Object UI in just a few minutes.

## Installation

Install Object UI packages using your preferred package manager:

```bash
pnpm add @object-ui/react @object-ui/components
```

## Basic Usage

Create your first Object UI component:

```tsx
import { ObjectUIRenderer } from '@object-ui/react';

const schema = {
  type: 'Container',
  props: {
    className: 'p-4'
  },
  children: [
    {
      type: 'Text',
      props: {
        value: 'Welcome to Object UI!',
        className: 'text-2xl font-bold'
      }
    }
  ]
};

export default function App() {
  return <ObjectUIRenderer schema={schema} />;
}
```
