## Usage

Customize e-commerce blocks for your store:

### Add Product Data

```json
{
  "type": "card",
  "dataSource": {
    "api": "/api/products",
    "method": "GET"
  }
}
```

### Add to Cart Action

```json
{
  "type": "button",
  "label": "Add to Cart",
  "action": {
    "type": "submit",
    "endpoint": "/api/cart",
    "method": "POST"
  }
}
```

### Customize Pricing Display

```json
{
  "type": "text",
  "content": "$299.99",
  "className": "text-2xl font-bold text-primary"
}
```
