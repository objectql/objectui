## Usage

All authentication blocks can be customized by modifying the schema. Common customizations include:

### Change Colors

```json
{
  "type": "button",
  "className": "bg-blue-600 hover:bg-blue-700"
}
```

### Add Data Sources

```json
{
  "type": "button",
  "label": "Sign In",
  "action": {
    "type": "submit",
    "endpoint": "/api/auth/login",
    "method": "POST"
  }
}
```

### Customize Layout

```json
{
  "type": "card",
  "className": "max-w-lg p-8"
}
```
