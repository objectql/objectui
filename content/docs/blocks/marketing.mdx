## Usage

Customize marketing blocks for your campaigns:

### Update Colors

```json
{
  "type": "card",
  "className": "bg-gradient-to-r from-blue-600 to-purple-600"
}
```

### Add Analytics

```json
{
  "type": "button",
  "label": "Get Started",
  "action": {
    "type": "analytics",
    "event": "cta_clicked"
  }
}
```

### Customize Content

```json
{
  "type": "text",
  "content": "Your custom headline here",
  "className": "text-4xl font-bold"
}
```
